# 霓虹俄罗斯方块游戏 - 规格说明

## 1. 游戏布局
- **游戏区域**: 左侧，10列 x 20行格子
- **信息面板**: 右侧，包含：
  - 下一个方块预览
  - 当前分数
  - 最高分
  - 消除行数
  - 当前等级
  - 最下方显示操作说明

## 2. 计分系统
- 消除1行: 1分
- 消除2行: 4分
- 消除3行: 9分
- 消除4行: 16分
- 连续消除：消除得分 * 连续次数^2，如消除了4行，是第三次连击，则分数为16 * 3^2 = 144分
- 每等级分数 = 上一等级 × 2

## 3. 升级系统
- 每消除10行升级一次
- 等级越高，下落速度越快

## 4. 视觉效果
- **霓虹灯管效果**: 方块边缘发光，颜色鲜艳
- **炫彩拖尾**: 下落时留下渐隐轨迹
- **消除特效**: 消除行闪烁消失，越多行越炫酷
- **升级特效**: 
  - 全屏彩虹闪烁2秒
  - 全屏震动特效1秒
  - 当前等级震动1秒

## 5. 控制
- 方向键: 移动/旋转
- 空格: 快速下落
- P: 暂停游戏
